Win32 MSDEV e*ORB X86 Build Configuration Release Notes

Version: 1.0.0

Release Date: December 2012

System Requirements

Host Operating System Target Operating System Supported Processors
Windows (XP/7) Windows (XP/7) X86

Environment Variables (All Targets)

The following environment variables must be set before starting Spectra CX.

  1. EORBHOME - set to the directory where e*Orb is installed.

Usage Instructions

Before starting Spectra CX, ensure that the PATH includes the directory containing the MSDEV compiler tool chain.

The following changes have to be made to components modeled in Spectra CX to generate correct XML for deployment:

  • Set the processor dependency name to x86
  • Update the processor_name property of the Devices and DeviceManagers to the above value.
  • Set the os dependency name to Windows
  • Update the os_name_name property of the Devices and DeviceManagers to the above value.
