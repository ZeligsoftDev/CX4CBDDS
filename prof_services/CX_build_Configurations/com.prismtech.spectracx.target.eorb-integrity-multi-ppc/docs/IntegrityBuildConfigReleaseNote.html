Environment Variables

The following environment variables must be set for all build configurations before starting Spectra CX.

  1. EORBHOME - set to the directory where Spectra ORB is installed.
  2. EORBTARGETENV - set to the EORBENV value of the target ORB.
  3. INTEGRITY_BSP - set to the name of the BSP.
  4. INTEGRITY_BASE - set to the base location of the Integrity install.
  5. INTEGRITY_COMP_DIR - set to the install location of the Multi compiler.

Usage Instructions

The following settings are required for components modeled in Spectra CX to generate correct XML for deployment:

  • The SCA code properties of each component implementation should be set to:
    • Filename: ensure this ends in .mod
  • The OS dependency name should be set to INTEGRITY and the correct version
  • The processor dependency name should be set to PPC
  • The os_name, os_version and processor_name properties of DeviceManagers, LoadableDevices and ExecutableDevices should match the above values.
