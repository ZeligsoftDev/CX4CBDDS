VxWorks 6.9 Spectra ORB ARM Build Configuration Release Notes

Version: 3.5.4

Release Date: February 2014

System Requirements

Host Operating System Windows XP
Target Operating System VxWorks 6.9.0
Supported BSPs TI DM81XX_EVM

Environment Variables

The following environment variables must be set before starting Spectra CX.

  1. EORBHOME - set to the directory where Spectra ORB is installed.
  2. EORBTARGETENV - set to the following value: vxworks-69-gnu-arm-win32

Usage Instructions

Before starting Spectra CX, ensure that the WindRiver environment has been set up.

The following changes have to be made to components modeled in Spectra CX to generate correct XML for deployment:

  • Change the SCA code properties of each component implementation to the following:
    • Filename: ensure this ends in .vxe
  • Set the OS dependency name to VxWorks and version to 6.8.0
  • Set the processor dependency name to ARM
  • Update the os_name, os_version and processor_name properties of the Devices and DeviceManagers to the above values.
