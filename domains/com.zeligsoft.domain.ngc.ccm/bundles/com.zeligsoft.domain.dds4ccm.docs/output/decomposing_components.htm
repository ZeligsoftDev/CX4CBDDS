Decomposing Components

This tutorial assumes that you have completed Tutorial 1: Getting Started and Tutorial 2: Top-Down_Design.

You can view a video of this tutorial here.  Depending on your screen size, you may want to close the Contents frame.

In this tutorial you will learn how to decompose the Philosopher component by adding a Brain and Hand Managers to the design.

Add Assembly Implementation and Diagram to the Philosopher

You need to add an Assembly Diagram to the existing Philosopher component.

To add an Assembly Implementation, follow these steps:

  1. Right-click the Philosopher component and from the context menu select Add CX > Assembly Implementation.

  2. Rename the assembly Phil_asm.

  3. Right-click Phil_asm and from the context menu select Add Diagram > Composite Structure Diagram.

  4. Rename the diagram Phil_asm.

  5. Note that the assembly inherits the ports from the Philosopher component.

  6. Save your model.

Create Sub-component Parts

The Philosopher needs a Brain and managers for its left and right hands.

To create sub-component parts, follow these steps:

  1. Using the Palette, drag-and-drop a CCMPart onto Phil_asm.

  2. Name the part Brain.

  3. Add two more parts: LH_Manager and RH_Manager.

  4. Save your model.

Design HandManager

To define a HandManager, follow these steps:

  1. Right-click the Philosopher package and from the context menu select Add CX > CCM Component.
    The tool displays the New Component wizard.

  2. Name the component HandManager.

  3. Ensure Create Assembly is de-selected.

  4. Click Finish.
    The tool creates the component.

  5. Save your model.

Add an Interface Port to HandManager

The HandManagers have to communicate with the forks through an interface port.

To add an Interface Port, follow these steps:

  1. From the Palette, drag-and-drop an Interface Port onto HandManager.

  2. Name the port fork.

  3. In the Properties tab, click (Edit Porttype).
    The tool displays the Element Specification Dialog.

  4. From the Matching items list, select ForkControl - DiningPhilosophers::objects::interfaces::ForkControl.

  5. Click OK.

  6. In the Properties tab, check Is Conjugated.

  7. Save your model.

Add HandManagers to Assembly

To add HandManagers to the assembly, follow these steps:

  1. Return to the Phil_asm diagram.

  2. Select both the manager blocks.

  3. In the Properties tab, click (Edit Definition).
    The tool displays the Element Specification Dialog.

  4. Set the type to be HandManager - DiningPhilosophers::components::HandManager::HandManager.

  5. Click OK.

  6. Save your model.

Connect the fork Ports

To connect the ports, follow these steps:

  1. From the Palette, drag-and-drop a Connection from the fork port on LH_Manager to the external leftHand port on Phil_asm.

  2. From the Palette, drag-and-drop a Connection from the fork port on RH_Manager to the external  rightHand port on Phil_asm.

  3. Save your model.

You now have HandManagers that can pick up and put down forks.

Define the Brain

To add a new component, Brain, follow these steps:

  1. Right-click the Philosopher package and from the context menu select Add CX > CCM Component.
    The tool displays the New Component wizard.

  2. Name the component Brain.

  3. Ensure Create Assembly is not checked.

  4. Click Finish.
    The tool creates the component.

  5. Save your model.

Add New Interface for Brain

The Brain needs a way to control the hands.

To add a new interface, follow these steps:

  1. In the Project Explorer, open objects.

  2. Right-click interfaces and from the context menu select Add CX > Interface.

  3. Rename the interface HandControl.

  4. Save your model.

Add HandControl to the Brain

To add new ports to Brain, follow these steps:

  1. From the Project Explorer, drag-and-drop HandControl onto the Brain.

  2. In the Properties tab, rename the interface leftHand.

  3. In the Properties tab, check Is Conjugated.

  4. From the Action toolbar, select Add Interface Port.

  5. Rename the port rightHand.

  6. In the Properties tab, check Is Conjugated.

  7. In the Properties tab, click (Edit Porttype).
    The tool displays the Element Specification Dialog.

  8. From the Matching items list, select HandControl - DiningPhilosophers::objects::interfaces::HandControl.

  9. Click OK.

  10. In the Properties tab, check Is Conjugated.

  11. Save your model.

The Brain's leftHand and rightHand control ports now implement the HandControl interface.

Add status port to Brain

To add a status port to Brain, follow these steps:

  1. In the Brain diagram, select the component Brain.

  2. From Palette, drag-and-drop an select Interface Port onto Brain.

  3. Rename the port status.

  4. In the Properties tab, click (Edit Porttype).
    The tool displays the Element Specification Dialog.

  5. From the Matching items list, select DDS_Write - DiningPhilosophers::connections::PhilStatus_conn::DDS_Write.

  6. Click OK.

  7. Save your model.

The Brain can now publish its status.

Define Brain type

To define the Brain type, follow these steps:

  1. Return to the Phil_asm diagram.

  2. Select Brain.

  3. In the Properties tab, click (Edit Definition).
    The tool displays the Element Specification Dialog.

  4. From the Matching items list, select Brain - DiningPhilosphers::components::Brain::Brain.

  5. Click OK.

  6. Save your model.

Connect status port on Brain to Phil_asm status port

To connect the ports, follow these steps:

  1. From the Palette, drag-and-drop a Connection from the status port on Brain to the status port on the Phil_asm.

  2. Save your model.

Add control ports to LH_Manager and RH_Manager

To add a control port to the manager blocks, follow these steps:

  1. In the Phil_asm diagram, select LH_Manager and RH_Manager.

  2. From the Action toolbar, select Add Interface Port.

  3. In the Properties tab, click (Edit Component).
    The tool displays the Element Specification Dialog.

  4. From the Matching items list, select HandControl - DiningPhilosophers::objects::interfaces::HandControl.

  5. Click OK.

  6. Rename the Interface Port control.

  7. Save your model.

Connect control ports to LH_Manager and RH_Manager to leftHand and rightHand Brain ports

To connect the ports, follow these steps:

  1. From the Palette, drag-and-drop a Connection from the rightHand port on Brain to the control port on RH_Manager .

  2. From the Palette, drag-and-drop a Connection from the leftHand port on Brain to the control port on LH_Manager .

  3. Save your model.

Tutorial Summary

In this tutorial, you learned how to decompose a component by modeling it as an assembly of components.
