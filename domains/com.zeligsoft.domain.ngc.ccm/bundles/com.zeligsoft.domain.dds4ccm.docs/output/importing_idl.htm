Importing IDL

This tutorial shows you how to re-use previously-modeled objects and interfaces by importing the IDL.

You can view a video of this tutorial here.  Depending on your screen size, you may want to close the Contents frame.

Create New Model

To create a new model, follow these steps:

  1. Select File> New > New DDS4CMM Project.
    The tool opens the New DDS4CCM Project wizard.
  2. In the Project name and Model name fields, type DiningPhilosophers2.
  3. Click Finish.

Import Model Library

You need to import the DCPS Model Library.  Follow the steps here.

Import IDL

To import IDL, follow these steps:

  1. Right-Click on the model and select Import...

  2. Choose the import source CCM > IDL Import.
    The tool opens the IDL Import dialog.

  3. Browse to the Source Folder and select only the IDL itself.

  4. In the Into Target Model field, click Browse... and navigate to the model.

  5. Click OK.

  6. In the Files to excludes list, type Components.idl and ccm_dds.idl.

  7. Click Finish.
    The tool opens the Importing IDL dialog, which includes a progress bar.

Once the importation is complete, the tool displays the IDL_Import_Results folder under the model in the Project Explorer.

Add a Component Diagram

To add a component diagram, follow these steps:

  1. Right-click IDL_Import_Results > Philosopher.

  2. From the context menu, select Add Diagram > CX Component Diagram.

  3. Rename the diagram PhilosopherDiagram1.

  4. Drag-and-drop the diagram onto the main tab.
    The tool renders the diagram.
